# 运算符

>对于`i++`和`i--`这样的用法，如果`i`是一个指针，那么有如下含义

- 对于`*++pt`：先将`++`应用于`pt`，再将`*`应用于递增后的`pt`，这意味着这个值代表的是下一个地址代表的值
- 对于`*pt++`：先将`*`应用于`pt`，得到`pt`指向的值，返回这个值，再对`pt`这个指针进行递减操作，指向前一个地址

###### `!`逻辑`NOT`运算符

- 介绍：`!`是**逻辑非运算符**，也称为**取反运算符**
- 作用：对操作数的逻辑值取反
- 结果类型：总是`bool`类型，值为`true` 或`false`
- 语法：`!experssion`

举例：C++将任何非零值视为`true`，将零视为`false`

>[!code]-
>```C++
>#include <iostream>
>using namespace std;
>int main()
>{
>	int x = 10,y = 0;
>	cout<<"x的值："<<x<<"\t"<<"y的值："<<y<<"\t"<<endl;
>	cout<<"x的取反值："<<!x<<"\t"<<"y的取反值："<<!y<<"\t"<<endl;
>	//以上是对数值的处理，接下来对指针处理
>	
>	int* ptr1 = nullptr;//将ptr设置为空指针
>	int* ptr2 = new int(5);
>	
>	cout<<"ptr1空指针的值："<<ptr1<<"\t"<<"ptr2的值"<<ptr2<<"\t"<<endl;
>	//空指针被视为 false，有效指针被视为 true
>	cout<<"ptr1空指针取反的值："<<!ptr1<<"\t"<<"ptr2取反的值"<<!ptr2<<"\t"<<endl;
>	
>	//以上是对指针进行处理，可用于判断指针是否为空
>	delete ptr2;
>	return 0;
>}
>```

以下是输出结果：

```text
x的值：10       y的值：0
x的取反值：0    y的取反值：1
ptr1空指针的值：0       ptr2的值0x7d1520
ptr1空指针取反的值：1   ptr2取反的值0
```

对于指针变量，我们对它使用`!`可以用来判断是否为空指针
- 如果是空指针，例如使用`!ptr`，会返回1，对应有值的指针会返回0

>`?:`运算符

- 介绍：(`?:`)是条件运算符，是唯一一个需要三个操作数的运算符
- 作用：类似`if-else`

举例：
```C++
5 > 3 ? 10 : 12//如果5大于3，返回10，否则返回12
//特殊的，该运算符是从右到左结合
//下面这个是一个例子
5 < 3 ? b : 5 > 3 ? 4 : 5//该代码和下面的作用一样
5 < 3 ? b : (5 > 3 ? 4 : 5)
//先计算括号里的，返回4；再计算外面的，因为5<3为假，因此返回4
```


>[!tip]-
>对于逻辑运算符，在C++标准库中有另一种表达方式：
>```mermaid
>flowchart LR
>	id1([与运算符：&&]) --> id2([and])
>	id3(["或运算符：||"]) --> id4([or])
>	id5([非运算符：!]) --> id6([not])
>```

# 跳转`goto`

```cpp
#include <iostream>
using namespace std;

int main() {
   int x = 10;
   
   goto jump;  // 错误：跳过 y 的初始化
   
   int y = 20; // y 在这里定义和初始化
   
jump:
   cout << "x = " << x << endl;
   // cout << "y = " << y << endl; // 错误：y 可能未初始化
   
   return 0;
}
```

我们可以使用`goto`语句跳转到`jump`语句处的代码，但我们并不推荐使用，因此介绍到此为止

# 字符库函数 `cctype`

该库函数中包含**字符分类函数**和**字符转换函数**

- **字符分类函数**
	- `isalpha(c)`  ：是否是字母 (a-z, A-Z)
	- `isdigit(c)`  ： 是否是数字 (0-9)
	- `isalnum(c)`  ： 是否是字母或数字
	- `islower(c)`  ： 是否是小写字母
	- `isupper(c)`  ： 是否是大写字母
	- `isspace(c)`  ： 是否是空白字符 (空格、制表符、换行符等)
	- `isblank(c)`  ： 是否是空白字符 (空格、制表符) - C++11 引入。。。
- **字符转换函数**
	- `tolower(c)`  ： 转换为小写（如果原本就是小写或不是字母，则不变）
	- `toupper(c)`  ： 转换为大写（如果原本就是大写或不是字母，则不变）


# 简单文件输入/输出

- 介绍：`fstream`是基于 `<iostream>` 的流类，提供了文件读写功能
	- **主要类：**
		- `ifstream` - 输入文件流（用于读取文件）
		- `ofstream` - 输出文件流（用于写入文件）
		- `fstream` - 文件流（既可读又可写）
- 要执行文件的输入输出操作需要使用库函数`fstream`
- 将`ofstream`对象与文件关联起来，可以使用`open()`方法
	- 使用该方法在使用完文件后，应使用`close()`方法将其关闭

>下面对该库函数中包含的对象进行简单介绍

- 在使用库里包含的函数之前，我们需要手动声明一个`ofstream`等类的对象，例如：
	- `ofstream outFile//outFile这个名字是自己定义的，你可以改成任意名字，只要便于标识`
	- 理论上可以只用一个标识符来实现一个类里所有的功能，但为了更好的可读性，还是为每一个功能定义一个独特的名字

以下是相关代码：**用于创建与写入文件(`ofstream`)**
>[!code]-
>```cpp
>include <iostream>
>include <fstream>
>include <string>
>using namespce std;
>int main()
>{
>	ofstream outFile("example.txt");
>	//创建一个名为example.txt的文件，并与之关联
>
>	ofstream fout;
>	fout<<"Hello World!"<<endl;
>	fout<<"这是第二行"<<endl;
>	//向example.txt中写入数据
>	
>	outFile.close();
>	//关闭文件，虽然程序停止运行时会自动关闭，但最好还是养成手动关闭的习惯
>	return 0;
>}
>```

上面的代码创建了一个`example.txt`的文件，并向里面写入了一些数据

- `open()`：不创建文件，只是与之关联
- `ofstream outFile("file.txt");`
	- 该代码在`example.txt`文件**存在**的时候会**清空里面的内容**，然后写入
	- **不存在**的时候会创建新文件
- `ofstream outFile("file.txt", ios::app);`
	- 该代码在文件存在的时候会在文件末尾追加内容
	- 文件不存在的时候会创建新文件

<br>

以下是相关代码：**用于读取文件(`ifstream`)**

上面的代码读取了一个`example.txt`文件，并逐行输出
