# 介绍：`fstream`是基于 `<iostream>` 的流类，提供了文件读写功能

- **主要类：**
	- `ifstream` - 输入文件流（用于读取文件）
	- `ofstream` - 输出文件流（用于写入文件）
	- `fstream` - 文件流（既可读又可写）
- 要执行文件的输入输出操作需要使用库函数`fstream`
- 将`ofstream`对象与文件关联起来，可以使用`open()`方法
	- 使用该方法在使用完文件后，应使用`close()`方法将其关闭

>下面对该库函数中包含的对象进行简单介绍

- 在使用库里包含的函数之前，我们需要手动声明一个`ofstream`等类的对象，例如：
	- `ofstream outFile//outFile这个名字是自己定义的，你可以改成任意名字，只要便于标识`
	- 理论上可以只用一个标识符来实现一个类里所有的功能，但为了更好的可读性，还是为每一个功能定义一个独特的名字

以下是相关代码：**用于创建与写入文件(`ofstream`)**
>[!code]-
>```cpp
>include <iostream>
>include <fstream>
>include <string>
>using namespce std;
>int main()
>{
>	ofstream outFile("example.txt");
>	//创建一个名为example.txt的文件，并与之关联
>
>	ofstream fout;
>	fout<<"Hello World!"<<endl;
>	fout<<"这是第二行"<<endl;
>	//向example.txt中写入数据
>	
>	outFile.close();
>	//关闭文件，虽然程序停止运行时会自动关闭，但最好还是养成手动关闭的习惯
>	return 0;
>}
>```

上面的代码创建了一个`example.txt`的文件，并向里面写入了一些数据

- `open()`：不创建文件，只是与之关联
- `ofstream outFile("file.txt");`
	- 该代码在`example.txt`文件**存在**的时候会**清空里面的内容**，然后写入
	- **不存在**的时候会创建新文件
- `ofstream outFile("file.txt", ios::app);`
	- 该代码在文件存在的时候会在文件末尾追加内容
	- 文件不存在的时候会创建新文件

<br>

以下是相关代码：**用于读取文件(`ifstream`)**
>[!code]-
>```cpp
>include <iostream>
>include <fstream>
>include <string>
>using namespce std;
>int main()
>{
>	ifstream inFile("exampel.txt");
>	string line;
>	while(getline.(inFile,line));
>	{
>		cout<< line << endl;
>		//逐行输出文件中的内容
>	}
>
>	if(inFile.eof())
>	{
>		cout<<"读取到文件末尾"<<endl;
>	}
>	inFile.close();
>	return 0;
>}
>```

上面的代码读取了一个`example.txt`文件，并逐行输出
