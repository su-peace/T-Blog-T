# 入门

对于爬虫入门，需要熟知爬虫的原理，知道，并会使用一些基本的爬虫相关库的函数

以下是一些会用到的库：
- 处理静态页面
	- `requests`
	- `BeautifulSoup`
- 复杂动态页面
	- `Selenium/Playwright`
- 大型爬虫项目
	- `Scrapy`
- API数据采集
	- 调用`requests`
- 高并发需求
	- `aiohttp `
	- `asyncio`

==对于初学者而言，需要优先学会使用`requests`和`BeautifulSoup`==
