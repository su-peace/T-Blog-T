# 简要介绍

- 在Python中，字典是一系列**键-值对**，我们可以使用键来访问与之关联的值，该值可以是任意类型的

- 字典用放在花括号`{}`内的一系列键-值对来表示
例如：`data = {'color': 'green','points':5}`

###### **访问**：

- `print(data['color'])`
这样将会输出`'color'`对应的值`green`

###### **添加**：

```python
data = {'color': 'green','points':5}
data['x_position'] = 0
data['y_position'] = 25
print(data)
```

输出：
```text
{'color': 'green', 'points': 5, 'x_position': 0, 'y_position': 25}
```
如上所示，数据被成功地加入到`data`字典中

###### 修改：

- `data['color'] = 'yellow'`
这行代码会把原来的`color`对应的元素改为`yellow`

###### **删除**：

- `del data['points']`
该代码会将`data`的`points`包括他对应的数值给删除

# 进阶

>[!code]- 
>```python
>data = {
>	'username':'admin',
>	'first':'donk',
>	'last':'fuck',
>	}
>for key,value in data.items():
> 	print("\nKey:" + key)
> 	print("Value:" + value)
>```
>
>输出：
>```text
>Key:username
>Value:admin
>
>Key:first
>Value:donk
>
>Key:last
>Value:fuck
>```

- `items()`方法：`data.items()`，该操作会返回一个键值对列表、
- `keys()`方法：`data.keys()`，在不需要用到字典中键对应的值的时候，可以使用这个方法，该方法**返回所有键值的列表**，（可省略）

>但一般默认情况下，遍历字典时，会默认遍历所有的键，因此，在不加`keys()`的时候，输出结果不变

- `values()`方法：`data.values()`，和`keys()`方法类似，但是，和`keys()`不同的是，他返回的是**所有键对应的值的列表**，不包含任何键
- `set()`函数：`set(data.values())`，如果不需要重复项，那么可以使用`set()`函数，该函数返回一个没有重复值的列表
