### 一些规则

1. 变量命名：
	- 变量名**只能包含字母、数字和下划线**，变量名可以字母或下划线打头，但**不能以数字打头**
	- 变量名不能包含空格，但可使用下划线来分隔其中的单词
	- 不要将Python关键字和函数名用作变量名

2. 字符串的使用： 在用单引号括起的字符串中，如果包含撇号，就将导致错误

# 语法与部分后续内容

1. 首字母大写：`变量名.title()`
>[!example]-
>```Python
>name="alpha"
>print(name.title())
>```

类似的还有`upper`，`lower`,分别代表全部大写和全部小写
以上操作**不会更改变量中存储的值**


2. 合并（拼接）字符串：Python借助来合并字符串 
>[!example]-
>```Python
>data1 = "Hello"
>daet2 = "world!"
>full_data = data1 + " " + data2#把data1，一个空格和data2进行合并处理
>print(full_data)
>```
 >输出结果应为
 >`Hello world!`


3. 将数值转换为字符串类型，使用`str()`
>[!example]-
>```Python
>num=45
>message="This number is" + str(num)
>print(message)
>```


4. **列表**：在Python中，列表是由一系列按特定顺序排列的元素。在Python中使用方括号`[]`来  表示列表，并用逗号来进行分割各个列表元素，例如以下代码
>[!example]-
>```Python
>list = ['this is one','two']
>print (list)
>```
>
>这段代码的输出为
>
>```
>['this is one','two']
>```

显然，这并不是我们期望的输出，下面我们对他进行优化

- **访问列表元素**：列表是一个有序的集合，因此我们只需要加上需要输出的那个内容所处的次序即可。需要注意的是**在计算机中第一个数字并不是1，而是0**。在Python中为访问**最后一个元素**，将其**次序定义为-1**，因此倒数第二个为-2，以此类推

- **修改、添加和删除元素**：
	- **修改**：先选择要修改的元素，再对元素进行重新赋值
	- **添加**
		- 在末尾添加：`列表变量名`.append`(要添加的元素)`、
		- 在中间添加：`列表变量名`.insert`(添加的元素所处的位置序号,添加的元素)`、
	- **删除**
		- 仅删除：`del 元素所处的位置`
		- 删除并使用：`列表变量名.pop()`[^7]，要选择某个位置的元素只需要在括号中输入对应的序号
		- 根据值来删除：`列表变量名.remove()`，这条代码的意思是查找括号内的元素在列表中的位置，并删除。同时它和`pop`一样，可以删除并使用


>[!example]-
>列表的输出
>```Python
>list1 = ['this is one ','two',3]
>print (list1[0],list1[1],list1[2])
>```
>将会输出
>`this is one  two 3`
>
>列表的修改、添加和删除
>```Python
>list1 = ['this is one ','two',3]
>list1[0] = 'first' #对list1[0]进行修改
>list1.append('fourth') #在list1的末尾添加元素fourth
>list1.insert(3,'between')#在第4个位置插入一个元素
>print (list1[:5])#输出经过处理list1列表
>#下面开始进行删除处理的实践
>del list1[-2]#删除倒数第二个元素
>popped_list1 = list1.pop()#将被删除的元素赋值给popped_list1，并且删除list1中的最后一个元素
>removed_list1 = list1.remove(5)#运行过程同上，只是该处理是通过搜索fourth元素进行删除
>print (list1)#输出经过上述全部过程后的list1
>```
>输出结果为
>```
['first', 'two', 3, 'between', 'fourth']
['first', 'two']
>```


5. **组织列表**：
	-  使用`sort()`可以对列表进行**永久排序**
		- 该方法将列表元素**按照首字母的顺序**进行排序
		- 如果要达到相反的效果只需要向`sort()`中传递`reverse=True`即可，如`num.sort(reverse=True)`
	- 使用`sorted()`进行**临时排序**
		- 可以按照特定顺序排列，但不影响原始数据的顺序
		- **使用方法**：`sorted(列表名)`
		- 类似的，也可以使用`reverse=True`对列表的顺序进行反向排序
	- 使用`reverse()`反转列表的元素
		- **使用方法**：`列表名.reverse()`
		- **注**：是反转列表里的顺序，不是字母顺序
		- 此方法是永久性的，直接更改列表的顺序
	- 确定列表的长度：`len()`
		- **使用方法**：`len(列表名)`

[^7]: 默认删除的是最后一个元素
