# 列表的应用：for循环
#for循环

>和C语言一样，Python中也有循环，并且，也叫`for`，但是，相比C语言中的`for`，Python中的更简单一点，下面将演示如何使用Python中的各种循环

>Pyhton中的for语法：
>```text
>for 变量 in 序列:
>	循环体
>```


###### 简单应用：

- 对于一个列表`first = ['one','two','three']`，我们要将其中的每一个元素都打印出来，那么可以使用下面这个语法

>[!code]- 
>```python
>first = [ 'one','two','three' ]
>for ch in first :
>	print(num)
>```
>以下是输出：
>```text
>one
two
three
>```

- `for num in first:`该语句将`first`中的元素一次一个地存到num中
- 当运行了一次上面的语句之后再运行`print(num)`就会把元素打印出来
- 当运行到列表中的最后一个元素的时候，该`for`循环自动终止，执行后面的语句

###### 稍微进阶：

- 使用`range()`函数，我们可以实现一些进阶操作，以下是代码：

>[!code]- 
>```python
>for value in range(1,5):#从1开始，到5结束
>	print(value)
>print('\n')
>#更进一步的，我们可以为range()函数添加步长
>for value in range(1,5,2):#从1开始，到5结束，中间间隔为2
>	print(value) 	
>```
>以下是输出
>```text
>1
>2
>3
>4
>
>1
>3
>```

- `range(10)`该函数只包含一个参数，意思是**从0开始，一直到5结束（不包含5），步长默认为1**
- `range(1,5)`函数也就相当于C语言的`for`循环括号内的`int i=1,i<5,i++`
	- 而修改步长也就是修改`i++`这一部分

- 如果要创建数字列表，可使用`list()`将`range()`的结果转换为列表
	- 例如：`num = list(range(1,6))`
- 而他的效果是将1到6之间的数字（不包括6）作为列表存储到`num`中

>在Python中，乘方运算符为`**`，具体用法为`3**2`，这个的意思就是3的二次方，即$3^2$

# 列表解析

<br>

###### 一个简单创建乘方列表的方法：


>[!code]- 
>```python
>num = [value**2 for value in range(1,11)]
>print(num)
>```
>
>下面是输出：
>
>```text
>[1, 4, 9, 16, 25, 36, 49, 64, 81, 100]
>```

- 和上述代码有相似之处的是以下代码：

>[!similar]- 
>```python
>num = []
>for value in range(1, 11):
 >   num.append(value**2)
>print(num)
>```

- 如果要使用第一种代码的书写方法，我们需要一个列表名`num`、方括号`[]`以及一个定义的表达式`value**2`，用于生成存储到列表的值。
- 然后需要一个`for`循环，用于提供用于表达式的值

# 使用列表的一部分

>在Python中，如何处理列表中的部分元素（Python中称之为切片），下面我们进行这一部分的学习

>[!code]- 
>```python
>#代码一
>num = ['one','two','three','four']
>print(num[0:2])#0是开始打印的索引，而2是结束的索引（不包含在输出结果里面）
>#代码二
>print(num[:3])#默认从0索引开始打印
>#代码三
>print(num[-1:])#输出最后一个元素
>#代码四
>print(num[::2])#从第0个元素开始，没两个元素一次输出
>```
>
>以下是输出：
>
>```text
>['one', 'two']
>['one', 'two', 'three']
>['four']
>['one', 'three']
>```

- 代码三中：如果把-1改成-3，那么他将打印最后三个元素，相应的，如果是`print(num[:-3])`将打印除了最后三个元素之外的元素
>[!tip]- 
>我们可以这样理解：`num[-1:]`从索引为-1的元素开始打印，然后把空着的默认为0，遇到0，默认退出这一进程类似C++中的`\0`

- 代码四中：2指的就是步长

### 切片处理

<br>

###### 遍历切片

>[!code]- 
>```python
>nums = ['one','two','three','four']
>for number in nums[:2]:
>	print(number.title())
>```
>
>以下是输出：
>
>```text
>One
>Two
>```

上面的代码将输出`nums`中的前两个元素，并大写，具体输出逻辑与上方代码一致

###### 复制列表

>[!code] 
>```python
>nums = ['one','two','three','four']  
>num2 = nums[:]#代码1
>#下面这个代码达成的效果和上面一行的代码一样，但他们有本质区别
>num2 = nums#代码2
>```

- 代码1是**切片拷贝**：
	- 代码1是创建了一个新的列表对象
	- 同时将`nums`存储的内容复制给`num2`
	- 他们的内存地址是不同的
- 代码2是**引用赋值（浅拷贝）**：
	- 如果之后修改`num2`，那么`nums`也会同步修改
	- 这种方法类似C中的指针，他们的内存地址相同，修改一个会把两个列表都修改

# 元组

Python将**不能修改的列表**称为**元组**

- 元组使用圆括号而不是方括号进行标识
	- 例如`data = (200,50)`
	- 我们甚至可以这样定义元组：`date = 200,50`，省略括号
- 我们**不能修改元组内部的元素**
	- 例如`data[0] = 250`是不被允许的
	- 但是我们**能修改整个元组变量**
		- 例如`data = (400,100)`这样的操作是被允许的
- 元组可包含任意类型
- 可以使用`tuple()`函数将可迭代的序列[^1]转换为一个元组

>[!note]- 对元组的操作
>```python
>t = (1, 2, 3)
>
># 索引访问
>print(t[0])    # 1
>
># 切片
>print(t[1:])   # (2, 3)
>
># 连接
>t2 = t + (4, 5)
>print(t2)      # (1, 2, 3, 4, 5)
>
># 重复
>print(t * 2)   # (1, 2, 3, 1, 2, 3)
>
># 成员检测
>print(2 in t)  # True
>
># 长度
>print(len(t))  # 3
>```

>[!元组解包]- 
>```python
># 基本解包
>a, b, c = (1, 2, 3)
>print(a, b, c)  # 1 2 3
>
># 交换变量
>x, y = 10, 20
>x, y = y, x     # 实际上是元组解包
>print(x, y)     # 20 10
>
># 使用*收集多余元素
>first, *middle, last = (1, 2, 3, 4, 5)
>print(first)    # 1
>print(middle)   # [2, 3, 4]  # 注意：middle是列表
>print(last)     # 5
>
># 函数返回多个值
>def get_user():
>    return "Alice", 25, "alice@example.com"
>
>name, age, email = get_user()  # 实际返回的是元组
>```


[^1]: 例如列表，字典之类的可迭代的序列
