==使用`json`模块来存储数据==

```python
import json

numbers = [1,2,3,4,5,7,11]

with open('example.json',`w`) as f:
	json.dump(numbers,f)
	
with open('example.json') as f:
	numbers = json.load(f)
```

- `json.dump()`：接受两个参数，要存储的数据以及可用于存储数据的文件对象
- `json.load()`：将列表读取到内存中，加载存储在`numbers.json`中的信息

# 重构

>**重构**是在不改变代码功能的前提下，改进代码的内部结构，使其更易于理解、维护和扩展。

**常见重构技巧**：
-  **提取函数/方法**：将原本的代码重新划分为多个函数，并进行一定层次上的优化
- **内联变量**：将函数调用替换为函数体
- **查询取代临时变量**：将临时变量提取为函数
- **分解条件表达式**：将复杂条件判断提取为独立函数
