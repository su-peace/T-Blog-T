# 删除空白：使用`rstrip()`

>[!example]- 
>```Python
>fuck="Python "#该字符串末尾存在一个空格
>print(fuck.rstrip)#删除空格，并输出
>```
>**注意**：这样的删除只是暂时的，实际`fuck`中储存的依旧是包含空格的字符串，如果要永久删除，需要对`fuck`的数据再次进行赋值
>```Python
>fuck="Python "#该字符串末尾存在一个空格
>fuck=fuck.rstrip()#删除空格，并进行再赋值处理
>print(fuck)
>```
>类似的还有除去开头空白的`lstrip()`和同时除去两端空白的`strip()`

# 对于数字列表的简单运用：

- 对于一个数字列表`num = [1,2,3,4,5,6]`
	- 使用`min(num)`可以找到`num`中的最小值
	- 使用`max(num)`可以找到`num`中的最大值
	- 使用`sum(num)`可以得到`num`中数字的总和

# Python中的循环语法

###### while循环：
基本语法：
```text
while 循环条件:
	循环体
```

示例：
```python
count = 0
sum = 0
while count<=100:
	sum += count
	count += 1
print(sum)
```

输出：
```text
5050
```
