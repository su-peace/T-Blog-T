# 介绍

- 模块是包含Python定义和语句的文件，用于组织和重用代码

- 模块内部包含已经编写好的函数语句，文件拓展名为`.py`
# 导入

- 在Python中有一个和`C++`中的头文件类似的东西：**模块**

>我们使用`import 模块名`来导入模块

- 如果要使用模块中的某一个函数：
	- `handle.add(a,b)`
	- 上面的例子是：使用`handle`模块中的`add`函数，并传递给`add`函数两个参数

>导入模块中的某个特定函数：

`from handle import add`

- 上面的代码含义是：导入`handle`模块中的`add`函数
- 在之后的使用中，我们就不需要使用模块拓展名了
	- 例如：原本是`handle.add(a,b)`，加入上面的代码之后是`add(a,b)`

>导入所有内容==(不推荐)==：

`from handle import *`

# 其他用法

- 给模块起别名：`import handle as ha`，将`handle`改名为`ha`
- 给模块中的某个函数起别名：
	- `from handle import add as plus`
- 如果要给模块中的多个函数改名：
	- `from handle import greet as hello , add as plus`
	- 类似使用C中的逗号表达式

# 函数编写指南

- 给形参指定默认值时，等号两边**不要有空格**
